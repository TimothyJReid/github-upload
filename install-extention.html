Vs Code Live Server Extention Setup

How to Install and Use VS Code's Live Server Extention

Install | Use

Installation Instructions

  1. In Visual Studio Code, click the Extentions icon in the left column toolbar.
  2. In the search bar that appears, type Live Server.
  3. Select the live Server extention. It should be the top choice.
  4. In the page for Li ve Server that appears, click the green Install button.

Usage Instructions

  • Visual Studio Code must be open to a folder to use Live Server.
  • In VS Code's Explorer, right click (windows) or Control+click (macOS) on a html file. From the context file that appears, select Open with Live server.
